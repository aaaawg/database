## 1. 주제
**전자책 서점 관리 시스템**   
구매를 통해 도서 이용이 가능하다.   
구매한 도서 가격의 3%를 포인트로 지급한다.
+ 도서 관리
  + 소장 도서와 대여 도서 구분
  + 소장: 모든 도서 가능, 대여: 일부 도서만 가능
+ 회원 관리
+ 주문 관리
+ 리뷰 관리
  + 구매자와 비구매자의 리뷰 구분
